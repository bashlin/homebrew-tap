# Bashlin Tap

## How do I install these formulae?

`brew install bashlin/tap/<formula>`

Or `brew tap bashlin/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bashlin/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
